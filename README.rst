8 coupons demo
==============

Installation
============

Install the app::

    $ pip install -r requirements.txt

Run application::

    $ make runserver

Run scraper::

    $ make runscraper

API endpoint::

    http://localhost:8080/games?search=<search string>



Requirements
============
* aiohttp
* motor
* requests
* snowballstemmer
* BeautifulSoup
